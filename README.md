# Grounding DINO Zira

---
Official pytorch implementation of [ZiRa](aa), a method for incremental vision language object detection (IVLOD)

## Install 
If you have a CUDA environment, please make sure the environment variable `CUDA_HOME` is set.

```bash
pip install -e .
```
